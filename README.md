### Скачивание и установка проекта на локальный компьютер:
1. Создать пустую папку для выкачивания в неё репозитория.
2. Открыть терминал начать вводить инструкции:
3. <code>cd ваша папка </code>
4. <code>git clone https://github.com/gramm2009/my-select-test-task.git</code>
5. <code>cd select-backend/</code>
6. <code>npm i</code>
7. <code>cd .. && cd select-frontend/ && npm i</code>

### Требования к программному обеспечению для работы приложений:
1. node -v === v22.12.0
2. npm -v === 10.9.0

### Пошаговая инструкция по запуску проекта:
1. Из корневой папки с вашим проектом вводите поочередно команды в терминале.
2. <code>cd select-backend/ && npm i</code>. Дождитесь установки пакетов зависимостей.
3. <code>cd select-backend/ && npm i</code>
4. <code>npm start</code>. Бекенд приложение запущено!
5. <code>cd .. && cd select-frontend/ && npm start</code>. Фронтенд приложение запущено!
