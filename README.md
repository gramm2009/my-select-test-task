# Разработано по ТЗ.
> ТЗ можно посмотреть в папке ТЗ. 

### Скачивание проекта на локальный компьютер:
1. Открыть терминал начать вводить команды:
2. <code>git clone https://github.com/gramm2009/my-select-test-task.git</code>
3. <code>cd my-select-test-task</code>

### Требования к программному обеспечению для работы приложений:
1. node -v === v22.12.0
2. npm -v === 10.9.0

### Пошаговая инструкция по запуску приложений:
* Находясь в папке *my-select-test-task* проекта, открыть 2 терминала начать вводить команды:
* Ввести в первый терминал для запуска Backend приложения команды:
> Установка пакетов: <code>cd select-backend/ && npm i</code>
> Запуск приложения: <code>npm start</code>
> Дождитесь установки пакетов зависимостей...
> Бекенд приложение запущено!

* Ввести во второй терминал для запуска Frontend приложения команды:
> Установка пакетов: <code>cd select-frontend/ && npm i</code>
> Запуск приложения: <code>npm start</code>
> Дождитесь установки пакетов зависимостей...
> Фронтенд приложение запущено!
